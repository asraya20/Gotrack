GoTrack 🌍

Crafting Journeys, Creating Memories.

GoTrack is a comprehensive, full-stack travel itinerary planner designed to help users organize their adventures seamlessly. From managing budgets to scheduling daily activities and tracking travel essentials, GoTrack serves as your digital companion for every trip.

✨ Features

Personalized Dashboard: View and manage all your upcoming and past trips in one earthy, themed interface.

Itinerary Timeline: Plan your days with precision. Add activities, transport legs, and accommodations.

Expense Tracker: Keep your travel budget under control with real-time expense logging.

Travel Checklist: Never forget your passport or charger again with a built-in packing and task list.

Secure Authentication: User registration and login systems to keep your travel data private.

Profile Management: Customize your user details and personal information.

🛠️ Tech Stack

Frontend: React.js, Axios, CSS3 (Custom Earthy Theme)

Backend: Django, Django REST Framework (DRF)

Database: PostgreSQL

Authentication: Token-based Authentication

🚀 Installation & Setup

Prerequisites

Python 3.x

Node.js & npm

PostgreSQL

Backend Setup

Navigate to the backend directory: cd backend

Create a virtual environment: python -m venv venv

Activate the environment: venv\Scripts\activate (Windows)

Install dependencies: pip install -r requirements.txt

Run migrations: python manage.py migrate

Start the server: python manage.py runserver

Frontend Setup

Navigate to the frontend directory: cd frontend

Install dependencies: npm install

Start the development server: npm run dev
